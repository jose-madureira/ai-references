# Guia de Referências sobre Inteligência Artificial

Olá, pessoal! Sejam bem-vindos ao meu repositório! Eu sou José Roberto Madureira Junior e, neste espaço, compilei um guia completo de materiais para quem deseja explorar temas essenciais no universo da Inteligência Artificial.

Este repositório reúne uma seleção de Livros e Cursos que abrangem desde os fundamentos até tópicos avançados, para ajudar você a desenvolver e aprofundar seus conhecimentos na área.
